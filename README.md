# AppCardapio
O presente projeto simula um cardápio de uma doceria especializada em festas tradicionais
de casamento, baile debutante e formaturas.

## Tech
Aplicativo em construção com React Native, com o intuito de estudar navegação entre telas. 
O tipo de navegação escolhida foi o Tab Navigation.
